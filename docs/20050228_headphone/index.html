今月の計画-WかAか-

2005.02.28 · diary, gadget, music, shopping

最近買い物にすら行けない日々です。
マンガも買えんCDも買えん。
当然の如くゲームなんて夢のまた夢です。
去年のヒキオタ生活が懐かしい(‘A`)

さて、そんな毎日なんでネタとしては・・・
・仕事の話
・電車の話
・欲しいものの話
・聴いてるものの話
くらいしかないわけですが。

今日は今月のお買い物計画！ サウンドボードとイヤホンは買ったので次は・・・
アンプとスピーカー！
のはずだったが・・・

ヘッドホンとヘッドホンアンプになりました。
アンプとスピーカーなんて、置き場がない＆大きい音もそうそう出せない
おまけに10万以上すっ飛んでいく・・・ということで断念。
手軽に大きな音出せるヘッドホンをいいやつに買い換えようかと。

今使ってるのが2つ、両方ともaudio-technica。
ATH-A9X(密閉型)
ATH-AD7(開放型)
の2種。
冬はA9X、夏はAD7と使い分けてたんですが、どちらかと言えば密閉型の
音が好みなので、密閉型で新しいのを買うことに。

で、候補が・・・
ATH-A1000
ATH-W1000
の二つ。なぜかまたオーテク。
SENNHEISERのHD25とかAKGのK271とかも考えたのだが・・・却下。
重かったり、つけてると頭痛くなりそうだった。そんな気がしただけですが！
あとは評判なんかを色々調べた結果・・・とでも言うべきか。

別に↑のオーテク2種が最高！オーテクマンセー！なんて評判はない。
おいらはロック/ポップス中心に最近ジャズ聞き出したりとかテクノも
ほんのりかじってみたりとか、とかく色々と手を出したがるのである。
んだもんで、「高音がいい」とか「低音がぼんぼん鳴る」とか味付けが
多いのも好きでない・・・出口はできるだけフラットなのが好きなのです。
あとは曲によってぐちゃぐちゃいじってやればいいわけであり。

そんなわけでフラットに近いというこの2機種。でもまあ、やっぱ違いは
あるわけで・・・
W1000→材質が木のせいか低音が抜けることが。高音が強め。
A1000→材質がチタンのせいか周囲の雑音が多いとハウジングが反響する。低音強め。
こんな特徴だそうだ。どちらも一長一短･･･コストパフォーマンスで言うと、
A1000がいい(W1000の方が最大で7,000円高い)のだが、微妙だ・・・
ちょっと視聴したくらいでわかるほど耳も肥えてないし、どうするべかなぁ。
今のところA1000に傾き気味。

ちなみに開放型は結構フラットだという評判のものも多かったのだが、開放型は
音漏れするので論外。AD7は音が籠もるのでそんなに好きではないのだが
さすがに二つ買う余裕はない・・・

さて、長くなった。ヘッドホンアンプについてはまた次回ー。

P.S.よく考えたら、値段的にSTAXのSRS2020とか3030買えちゃうんだよなぁ・・・いきなりはやめといたほうがいいか。
