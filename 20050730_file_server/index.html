ファイルサーバー

さて、キーボードを購入してきました。 Realforce101（PS/2）ML01 なぜか101英語キーボード。さっき接続を完了したのですが、配置が違ったり、日本語切り替えがAlt+`だったりで慣れないうちは使いづらそうです。 でも、それを補って余りあるのが･･･打ちやすさ。キーが軽い軽い♪ 英語配列が打ちづらいなら、106日本語配列買えばいいじゃん！！という声が聞こえてきそうですが、気にしない。こっちの方がなんかかっこよさそうじゃん！とか思って買ったのは秘密。 ･･･そこ、バカとか言わない_|￣|○ ちょっとでも早く慣れるためにも、ネタが続く限り毎日更新！！を目指そうかと。 はてさて、今日の本題はこれではないのです。

実はうち、デスクトップとノートが1台づつあるのです。 ノートは破格で手に入れたPen3-1GHzマシン。 もう2年くらいうちにあるのですが、ろくな使い方をされてません(;-_-) 最近HDDも手狭になってきたので、ファイルサーバー化することにしました。 元々Win2kが入っているのですが、このまま使っても面白くない。 というわけで･･･Linux導入！ ｢はじめてのFedora Core4｣という本を買ってきてお勉強中。 今現在インストールの真っ最中でございます。 予定としては･･･ ・FTPサーバー ・Webサーバー ・↑をとMySQL導入してMTを使ってみる ・ついでにPHP使ってBlog自作 てな感じです。 今書いた事って、超初歩的な事だなぁと実感。 サーバーの構築・管理・運用のステップがちゃんと踏めるようになるのはいつのことだろう･･･ 可能であれば身内公開する予定。一般公開するとえらいことになりそうで怖いので、知り合いに開放のみになるかなぁ･･･ まあ、当分は良い遊び道具になりそうです。 P.S.　キーボードのレポートは来週くらいに書くかも。 では?
